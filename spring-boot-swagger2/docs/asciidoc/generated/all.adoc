[[_deleteuserusingpost]]
==== 删除用户
....
POST /api/deleteUser
....


===== 说明
id必传


===== 参数

[options="header", cols=".^2a,.^3a,.^9a,.^4a"]
|===
|类型|名称|说明|类型
|**Body**|**id** +
__必填__|用户id|integer (int64)
|===


===== 响应

[options="header", cols=".^2a,.^14a,.^4a"]
|===
|HTTP代码|说明|类型
|**200**|OK|string
|**201**|Created|无内容
|**401**|Unauthorized|无内容
|**403**|Forbidden|无内容
|**404**|Page Not Found|无内容
|===


===== 消耗

* `application/json`


===== 生成

* `\*/*`


===== HTTP请求示例

====== 请求 path
----
/api/deleteUser
----


====== 请求 body
[source,json]
----
{ }
----


===== HTTP响应示例

====== 响应 200
[source,json]
----
"string"
----

=== UserEntity

|名称|说明|类型
|**id** +

|**userName** +
__可选__|用户名 +
**样例** : `"关羽"`|string
|**userSex** +
__可选__|用户性别 +
**样例** : `"男"`|string
|===
